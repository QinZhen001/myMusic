# myMusic

* vue2.x全家桶(包括vuex,vue-router)
* Css框架采用stylus
* 网络请求采用axios,jsonp
* 采用fastclick消除在移动浏览器上触发click事件与一个物理Tap(敲击)之间的300延迟
* 采用vue-lazyload图片懒加载
* 采用better-scroll移动端滚动库
* create-keyframe-animation
* 打包工具webpack
